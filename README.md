# RapunzelV2 React Native App

This is a React Native application built with Expo, featuring a complete development setup with modern tools and best practices.

## Documentation

All documentation has been moved to the [docs](./docs/) folder:

- [Main Project README](./docs/README.md) - Project overview, setup, and development guide
- [Building for Android](./docs/BUILDING_ANDROID.md) - Instructions for building Android APK
- [QWEN Summary](./docs/QWEN.md) - AI assistant summary of the project

The `.expo` directory is kept in the project root as it contains Expo-specific configuration files required for the development server.
The `.qwen` directory is kept in the project root as it contains AI assistant configuration files.
